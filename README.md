# docker-compose-app-db
Multi-container Docker Compose setup demonstrating a real-world backend system.

## Components
- Application Service
- PostgreSQL Database
- Internal Docker Network
- Persistent Volumes

## Architecture
App ↔ Database via Docker Network

## Run
```bash
docker compose up -d
```
